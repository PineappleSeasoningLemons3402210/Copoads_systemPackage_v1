# 🚀 Copoads

**Copoads** is a developer toolkit designed to **analyze, clean, and improve web code** (HTML, CSS, and JavaScript).
It helps streamline workflows, enforce best practices, and make editing websites faster and smarter.

---

## ✨ Features

* 🧹 Format and clean HTML, CSS, and JavaScript
* 🔍 Detect common coding issues and bad practices
* ⚡ Optimize structure and improve readability
* 🧠 (Optional) AI-powered code suggestions
* 🛠 Command-line interface (CLI) for fast usage
* 📦 Modular design for easy extension

---

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/copoads.git
cd copoads
```

Install dependencies:

```bash
npm install
```

---

## 🧑‍💻 Usage

Run Copoads from the command line:

```bash
node src/cli.js <path-to-project>
```

### Example:

```bash
node src/cli.js ./my-website
```
